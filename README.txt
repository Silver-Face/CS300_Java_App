Instructions: For Linux Users

1.	Download all ".java" files into a arbitrarily-named single directory.
2.	Open BASH terminal and go into directory.
3.	Compile with javac with "javac *.java".
4.	Run ServerTestingSuit with "java ServerTestingSuite"

Running once will create "UserList.txt" and a new directory "User_History" with subdirectories for every name on UserList.
